# MVP 任務清單

## Phase 1：專案基礎

- 建立 Next.js + TypeScript 專案
- 設定 Tailwind CSS
- 設定 Prisma + PostgreSQL
- 建立基本資料庫 schema
- 匯入阿蓮國中 114 學年教材版本資料

## Phase 2：帳號與角色

- 建立使用者資料表
- 支援學生、家長、班級管理者角色
- 建立學生加入班級流程
- 建立家長連結學生流程

## Phase 3：班級共用設定

- 建立班級
- 產生班級代碼
- 設定段考日期
- 設定各科範圍
- 設定老師目前進度

## Phase 4：個人作息與補習

- 設定平日到家與睡覺時間
- 設定晚餐、洗澡、自由時間
- 新增補習科目、星期、時間與通勤
- 設定補習後疲勞程度

## Phase 5：每日任務

- 產生今日任務
- 顯示任務預估時間
- 支援完成、部分完成、沒完成
- 支援未完成原因
- 建立簡單重排

## Phase 6：儀表板

- 學生首頁：今日任務與進度
- 家長首頁：本週完成率與作息提醒
- 班級管理首頁：段考範圍與班級弱點統計

## Phase 7：驗證與調整

- 用國一、國二、國三各建立一組測試資料
- 測試補習日排程
- 測試未完成重排
- 測試考前一週衝刺模式
